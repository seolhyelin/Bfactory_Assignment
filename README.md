## 🤍 과제 소감
Canvas 라이브러리를 처음 사용해봐서 어려움이 있었다. 주어진 시간 내에 완성하려다 보니 파일 구조를 효율적으로 나누지 못했다. 기능을 다 완성하지 못해서 아쉬움이 많이 남는다. 리팩토링하면서 추가 기능을 구현해보고, 가독성있는 코드로 만들어 봐야겠다. 그래도 이미지 툴을 만들어 볼 기회가 주어져서 너무 감사했다!
